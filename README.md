# safe-POMDP
CS4010 Algorithms for Planning &amp; Scheduling project D (Team of 4)


Source of safe-MDP for deterministic world: https://github.com/befelix/SafeMDP .
M. Turchetta, F. Berkenkamp, A. Krause, "Safe Exploration in Finite Markov Decision Processes with Gaussian Processes", Proc. of the Conference on Neural Information Processing Systems (NIPS), 2016. 


Extending the accompanying paper, the aim was to make it work in non-deterministic world. The approach was successful in polynomial time for certain toy examples.
